Agent Context Center

AI agents have AGENTS.md. Humans need dashboard.html.

Agent Context Center scans a repo's scattered AI context files and emits one self-contained, offline, deterministic dashboard.html that you can open, review, and commit. Markdown stays the source of truth for the agents; the dashboard is the map for the people. No server, no build step, no telemetry, stdlib Python 3.12+ only.

What it finds: CLAUDE.md, AGENTS.md, Cursor rules, skills, agents, hooks, commands, MCP servers, the markdown doc index, open - [ ] TODOs, and flat cross-references between them.
